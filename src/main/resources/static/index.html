구현 범위

  • Spring WebSocket + STOMP 실시간 메시지 송수신
  • 입장 / 퇴장 / 끊김 / 재연결 presence 처리
  • clientEventId 기반 중복 이벤트 방지
  • serverSequence 기준 deterministic replay
  • Snapshot + delta replay 기반 timeline 복원

주요 API

  • POST /sessions
  • POST /sessions/{id}/join
  • POST /sessions/{id}/events
  • GET /sessions/{id}/timeline?at=...
  • POST /sessions/{id}/snapshots
